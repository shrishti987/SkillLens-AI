🚀 InsightAI – Analyst in a Box

InsightAI is an AI-powered data analytics platform that automatically analyzes datasets and generates meaningful insights.  
It helps users explore data, detect patterns, visualize trends, and perform machine learning predictions with minimal effort.
The goal of this project is to make **data analysis accessible, automated, and intelligent**, even for users with limited technical knowledge.

🌟 Features

• 📊 **Dataset Summary Dashboard**  
Automatically generates dataset statistics like rows, columns, missing values, and data types.
• 🔍 **Automatic Target Detection**  
Identifies potential target columns for machine learning tasks.
• 📉 **Interactive Data Visualizations**  
Generate charts and graphs to understand patterns and relationships in the data.
• 🧩 **Missing Value Detection**  
Detects null values and displays them using a **heatmap visualization**.
• 🤖 **Machine Learning Predictions**  
Applies ML models to predict outcomes and analyze patterns.
• 📈 **Feature Importance Analysis**  
Highlights which features impact predictions the most.
• ⚡ **Auto Data Processing**  
Handles different datasets automatically without manual preprocessing.

📊 How It Works

1️⃣ Upload a dataset (CSV format)
2️⃣ InsightAI automatically analyzes the data
3️⃣ View dataset summaries and visualizations
4️⃣ Detect missing values and feature relationships
5️⃣ Run machine learning models for predictions
